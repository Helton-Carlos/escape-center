# Front-end: React.js + TypeScript + Vitest + Tailwind + DaisyUI

# Back-end: C# + .NET-8 + POSTGRESQL

**💬 About.**

Projeto feito loja de reparos automotivos.

**👇 Follow the steps.**

```bash
git clone git@github.com:Helton-Carlos/escape-center.git
```

**👇 Init.**

```bash
cd back-escape-center
```

```bash
cd dotnet run
```

```bash
cd ../
```

```bash
cd front-escape-center
```

```bash
npm install
```

```bash
npm run dev
```
